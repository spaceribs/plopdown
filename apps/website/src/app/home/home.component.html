Plopdown

The Video Annotating Web Extension

Plopdown is an extension which allows you to annotate any web video. It been tested and works on Youtube, Amazon Video, Netflix, and Vimeo.
After creating your annotations, you can download and distribute them to friends and followers by sending them a file or generating a plopdown:// URL.

It's currently in development and the alpha should be available soon!
